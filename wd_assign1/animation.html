Liquid Button Animation

By Nguyen Duc Tam

I have implemented an animation to the About Me button on the About page. This is the CSS code that I have used to make this animation, I will further explain this code down below

  1. #button: This selector targets an element with the ID "button" (presumably a button in HTML). It sets various styles like padding, display, text decoration, etc., to give it a certain appearance.
  2. a strong: This selector targets a strong element within an anchor (a) element. It sets styles like font family, weight, color, etc., for the text inside the button.
  3. a .liquid: This selector targets an element with the class "liquid" that is a child of an anchor (a) element. It's positioned absolutely to cover a portion of the button area. This is the liquid part that animates on hover.
  4. a:hover .liquid: This selector targets the liquid element when the parent anchor element is hovered over. It changes the position of the liquid, making it move upward (top: -120px;).
  5. a .liquid:before and a .liquid:after: These pseudo-elements create two additional elements within the liquid element. They are positioned absolutely to cover the entire area of the liquid. They are used to create the visual effect of the liquid. One has a white background (before), and the other has a green background (after).
  6. @keyframes animate: This defines an animation called "animate". It rotates the liquid elements from 0 degrees to 360 degrees around their center point. This gives the illusion of the liquid moving or swirling.
  7. Inside @keyframes animate, 0% and 100% specify the starting and ending states of the animation respectively.

In summary, this CSS code creates a button with a liquid effect that moves and swirls when hovered over. The liquid is represented by a greenish shape that rises slightly above the button's surface. The animation creates a rotating effect, giving the impression of flowing liquid.
