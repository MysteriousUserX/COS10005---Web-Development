Features

By Nguyen Duc Tam

Hamburger Menu

I have implemented an animation to the navigation menu on the top of the page. On mobile devices, the navigation menu will be replaced with a hamburger icon. When clicked or tapped, the hamburger menu expands to reveal a navigation menu, allowing users to access various parts of a website.

Current webpage highlight

An underline has been added to the navigation tab of the current page

"Same as delivery address" checkbox

This checkbox was added to the Order page. When the checkbox is selected, the billing address fields will automatically populate with the delivery address. If the delivery address remains incomplete upon checkbox selection, prompt an alert or an inline message stating: “Please enter your delivery address first”.

Credit Card Validation

Another function added to the Order page. Enable users to select among three credit card options: Visa, MasterCard, and American Express. Depending on the chosen credit card type, restrict the length of the credit card number to 16 digits for Visa and MasterCard, and 15 digits for American Express.
